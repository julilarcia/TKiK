| Typ tokenu | Przykład      | Opis            |
|------------| ------------- | --------------- |
| KEYWORD    | var, if, func | słowa kluczowe  |
| IDENTIFIER | x, suma       | nazwy zmiennych |
| NUMBER     | 123, 3.14     | liczby          |
| STRING     | "abc"         | tekst           |
| OPERATOR   | + - * / = > < | operatory       |
| SEPARATOR  | ( ) { } , ;   | separatory      |
| COMMENT    | // coś        | komentarz       |
| WHITESPACE | spacja        | odstępy         |
