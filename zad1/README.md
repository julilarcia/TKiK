# Rozpoznawane tokeny

| Nazwa tokenu | Kod tokenu | Opis |
|---|---|---|
| Liczba całkowita | `INTEGER` | ciąg cyfr od 0 do 9 |
| Identyfikator | `IDENTIFIER` | ciąg liter lub liter i cyfr, zaczynający się od litery |
| Dodawanie | `PLUS` | operator `+` |
| Odejmowanie | `MINUS` | operator `-` |
| Mnożenie | `MUL` | operator `*` |
| Dzielenie | `DIV` | operator `/` |
| Lewy nawias | `LPAREN` | znak `(` |
| Prawy nawias | `RPAREN` | znak `)` |
| Koniec wejścia | `END` | koniec skanowanego wyrażenia |
| Błąd | `ERROR` | niepoprawny znak |
